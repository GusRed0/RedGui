# RedGui
## Commands
- Fly
- Walkspeed: changes walkspeed
- Fling
- Target fling: you type someone's name to fling
- Walk-fling: fling, but no spinning
- Teleport to player
- Teleport to position: Vector3
- Click teleport: While holding any keybind you choose, wherever you click, you teleport
- Anti-fling: Disables everyone's collisions to prevent you from getting flinged
- Noclip: cross walls
- JumpPower: changes jump power
- Spectate: spectate someone (type the name)
- Vehicle fly: fly in a vehicle
- Rejoin
- ESP: creates a Highlight for everyone in the server, which allows you to see where everyone is
